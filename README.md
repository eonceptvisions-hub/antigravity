 ❄️ Outreach Brain | Glassmorphism OS

**Outreach Brain** is a high-vibrancy, AI-powered lead generation and outreach automation platform. Built with a premium **Subtle Glassmorphism** UI, it transforms raw data into actionable intelligence and personalized communication.

---

## ⚡ Core Features

### 🔍 Lead Prospecting
- **Global Research**: Automate lead discovery using **Gemini 2.0 Flash** with **Google Search Grounding**.
- **Industry Targeting**: Find decision-makers, hiring managers, or VCs in specific sectors.

### 🏛️ Intelligence Vault
- **Personnel Master Index**: Centralized hub for managing leads stored in `vault.csv`.
- **Auto-Enrichment**: Use OSINT protocols to find missing emails, websites, and professional details.
- **Deduplication**: Clean your database by filtering redundant nodes.

### 🧠 Neural Strategy & Drafting
- **AI Strategist**: Consult with the "Outreach Strategic Brain" to refine your mission parameters.
- **Personalized Synthesis**: Generate bespoke, low-friction cold emails based on indexed intelligence (PDFs, DOCX, Web Intel).

### 🚀 Mission Launch
- **Gmail API Support**: Secure, modern OAuth2 integration for broadcast-scale dispatch.
- **Legacy SMTP**: Fallback support for classic email authentication.
- **Real-time Analytics**: Track mission performance with a status-driven performance matrix.

---

## 🛠️ Tech Stack

- **Frontend**: [Streamlit](https://streamlit.io/) (Custom Glassmorphism CSS)
- **AI Engine**: [Google Gemini 2.0 Flash](https://ai.google.dev/)
- **Data Handling**: Pandas & CSV
- **Authentications**: OAuth2 (Google Cloud) & SMTP
- **File Processing**: PyPDF, python-docx

---

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have Python 3.9+ installed.

### 2. Installations
```bash
pip install -r requirements.txt
```

### 3. Setup API Keys
- Obtain a **Gemini API Key** from [Google AI Studio](https://aistudio.google.com/).
- (Optional) Configure **Google OAuth2** in the sidebar for Gmail API access.

### 4. Launch the App
```bash
streamlit run app.py
```

## 🎨 Design Philosophy
The app utilizes a **Glassmorphism v5.6** aesthetic, featuring:
- **Aurora Glass Backgrounds**: Dynamic, atmospheric visuals.
- **Backdrop Blur**: High-contrast overlays for readability.
- **Pill UI Components**: Modern, animated buttons and interactive nodes.
- **Typography**: Powered by the **Rubik** font family for a sleek, premium feel.

---

## 📋 Project Structure
- `app.py`: The primary UI and Streamlit entry point.
- `backend.py`: Core logic for lead search, AI drafting, and email dispatch.
- `requirements.txt`: Project dependencies.
- `vault.csv`: Local storage for leads.
- `improvements.md`: Tracked items and future roadmap.
